### Instructions :

1. **Dépendances**  
   - Assure-toi que **wmctrl** est installé sur ton système :  
     ```bash
     sudo pacman -S wmctrl
     ```
   - Installe le module **python-osc** :
     ```bash
     pip install python-osc
     ```

2. **Configuration de VRChat**  
   - Dans VRChat, active OSC via le menu Action → Options → OSC.
